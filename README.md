# СКФУ: Система учета общежития (Desktop MVP)

Прототип десктопного приложения для автоматизации учета проживания студентов в общежитии.

## Технологии
- Python 3.11+
- Tkinter (встроен в Python)
- SQLite (локальная БД)

## Функции MVP
- Авторизация и роли (admin / commandant / accountant / viewer)
- CRUD по студентам (добавление + поиск)
- Учет комнат и текущей занятости
- Заселение/выселение
- Финансовые начисления и оплаты
- Отчет по должникам

## Запуск
```bash
python3 main.py
```

Логин по умолчанию:
- `admin`
- `admin123`

## Структура
- `main.py` — точка входа
- `app/database.py` — инициализация и подключение SQLite
- `app/auth.py` — аутентификация и хэширование паролей
- `app/services.py` — бизнес-логика
- `app/ui.py` — интерфейс Tkinter
- `docs/architecture.md` — описание архитектуры и расширения
